# Website Project with SASS and Compass

## Description

This project is a static web page built with HTML5 and styled with CSS generated via SASS and Compass.  
The page features a header with logo and title, a navigation menu, several articles with images and texts, a thumbnails section, and a footer.

---

## Technologies Used

- HTML5  
- CSS generated with SASS & Compass  
- Local images stored in the `img/` folder

---

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install Compass and SASS if not already installed:

```bash
gem install compass
gem install sass
```

Compile the SASS files into CSS:

```bash
compass compile
```

---

## Features

- Responsive structure 
- Articles with images and metadata (author, title)  
- Thumbnails section with images and captions  
- Simple navigation menu  
- Footer
